# Throw Master

Pixel-art javelin rhythm runner with Supabase leaderboard and replay support.

## Deploy

This folder is the clean GitHub Pages release. Upload the contents of this folder to the repository root:

- `index.html`
- `.nojekyll`
- `README.md`

With GitHub Pages enabled for the repository root, the game opens at:

```text
https://<github-user>.github.io/<repo-name>/
```

The public Supabase URL and publishable key are embedded in `index.html`. Do not add service-role or secret keys to this repository.
